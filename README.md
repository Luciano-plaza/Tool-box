# 🧰 Tool-box

¡Bienvenido a **Tool-box**! Esta es una colección de herramientas escritas en Python 3 diseñadas para mejorar tu seguridad personal y practicar conocimientos básicos de análisis de red, esta caja de herramientas cuenta con:

- Generador de contraseñas seguras
- Analizador de entornos de red
- Escaner de puertos
- Desencriptador de contraseñas (hash SHA-256).

>[!WARNING]
> **Disclaimer:** Esta herramienta fue creada sólo con fines y uso educativos, no debe usarse para acciones ilegales o maliciosas

## 🚀 Inicio Rápido

>[!NOTE]
>Seguir las instrucciones de las herramientas para poder usarlas correctamente

### Requisitos previos
* Python 3.x instalado.
* Git (para clonar el repo).

### Instalación

---

1. Clona el repositorio:

```bash
git clone [https://github.com/Luciano-plaza/Tool-box.git](https://github.com/Luciano-plaza/Tool-box.git)
```

2. Entra al directorio:

```bash
cd Tool-box
```

3. Requirements

```
pip install -r requirements.txt
```

3. Ejecuta el script:

```bash
python3 Toolbox.py
```
